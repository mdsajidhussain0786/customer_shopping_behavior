## 📌 Project Overview

Understanding customer behavior is crucial for improving revenue and engagement.  

In this project, customer transactional data is analyzed to uncover:

- 📈 Revenue trends across categories  
- 👥 Customer segmentation  
- 🎯 Age-group based purchasing patterns  
- 📦 Subscription impact on sales  
- 🛒 Product category performance  

The objective is to transform raw data into meaningful insights that support strategic decision-making.
